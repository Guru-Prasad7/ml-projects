# 🌾 ML Projects

Two Machine Learning web applications built with Python and Streamlit.

---

## 🌿 Project 1: Crop Yield Predictor

Predicts crop yield based on environmental and agricultural inputs using Random Forest ML model.

### Features
- Predict yield for multiple crops
- Model Performance analytics
- Dataset Explorer with charts

### Tech Stack
- Python, Scikit-learn, Random Forest
- Streamlit, Plotly

🔗 **Live Demo:** https://c32631a6-b9e5-42e6-8eaa-413559edd19a-00-1bdoi3nsnnmj.pike.replit.dev/

---

## 🎓 Project 2: Student Performance Predictor

Predicts student pass/fail based on academic inputs using ML classification.

### Features
- Student performance prediction
- Interactive web UI

### Tech Stack
- Python, Scikit-learn
- Streamlit

---

## 🚀 How to Run
pip install -r requirements.txt
streamlit run app.py
